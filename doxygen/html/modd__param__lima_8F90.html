Detailed Description

MODD_PARAM_LIMA - declaration of the control parameters for use in the LIMA scheme.

PURPOSE

The purpose of this declarative module is to declare the microphysical constants. This includes the descriptive parameters for the raindrop and the parameters relevant of the dimensional distributions.

IMPLICIT ARGUMENTS

None
