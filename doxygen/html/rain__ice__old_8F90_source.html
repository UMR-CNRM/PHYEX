28!!**** * - compute the explicit microphysical sources
29!!
30!! PURPOSE
31!! -------
32!! The purpose of this routine is to compute the slow microphysical sources
33!! which can be computed explicitly
34!!
35!!
36!!** METHOD
37!! ------
38!! The autoconversion computation follows Kessler (1969).
39!! The sedimentation rate is computed with a time spliting technique and
40!! an upstream scheme, written as a difference of non-advective fluxes. This
41!! source term is added to the future instant ( split-implicit process ).
42!! The others microphysical processes are evaluated at the central instant
43!! (split-explicit process ): autoconversion, accretion and rain evaporation.
44!! These last 3 terms are bounded in order not to create negative values
45!! for the water species at the future instant.
46!!
47!! EXTERNAL

784 + 12.0 * zhlc_rcmax(:)**2 )

793 / ( (6.0 * (zhlc_rcmax(:))**2) &
794 - (24.0 * (zhlc_rcmax(:)) * zrcrautc(:)) &
795 + (12.0 * zrcrautc(:)**2) )

822 zhlc_hrc(:) = 0.0
823 ELSEWHERE
824 zhlc_lcf(:) = 0.
825 zhlc_hcf(:) = 0.0
826 zhlc_lrc(:) = 0.
