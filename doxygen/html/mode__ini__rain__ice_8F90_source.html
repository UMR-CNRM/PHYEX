12!
13!!**** *INI_RAIN_ICE * - initialize the constants necessary for the warm and
14!! cold microphysical schemes.
15!!
16!! PURPOSE
17!! -------
18!! The purpose of this routine is to initialize the constants used to
19!! resolve the mixed phase microphysical scheme. The collection kernels of
20!! the precipitating particles are recomputed if necessary if some parameters
21!! defining the ice categories have been modified. The number of small
22!! time steps leading to stable scheme for the rain, ice, snow and ggraupeln
23!! sedimentation is also computed (time-splitting technique).
24!!
25!!** METHOD
26!! ------
27!! The constants are initialized to their numerical values and the number
28!! of small time step is computed by dividing the 2* Deltat time interval of
29!! the Leap-frog scheme so that the stability criterion for the rain
30!! sedimentation is fulfilled for a Raindrop maximal fall velocity equal
31!! VTRMAX. The parameters defining the collection kernels are read and are
32!! checked against the new ones. If any change occurs, these kernels are
33!! recomputed and their numerical values are written in the output listing.
34!!
35!! EXTERNAL
36!! --------
37!! GAMMA : gamma function
38!!
