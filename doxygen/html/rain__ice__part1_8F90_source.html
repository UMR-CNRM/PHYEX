26!!**** * - compute the explicit microphysical sources
27!!
28!! PURPOSE
29!! -------
30!! The purpose of this routine is to compute the slow microphysical sources
31!! which can be computed explicitly
32!!
33!!
34!!** METHOD
35!! ------
36!! The autoconversion computation follows Kessler (1969).
37!! The sedimentation rate is computed with a time spliting technique and
38!! an upstream scheme, written as a difference of non-advective fluxes. This
39!! source term is added to the future instant ( split-implicit process ).
40!! The others microphysical processes are evaluated at the central instant
41!! (split-explicit process ): autoconversion, accretion and rain evaporation.
42!! These last 3 terms are bounded in order not to create negative values
43!! for the water species at the future instant.
44!!
45!! EXTERNAL
46!! --------
47!! None
48!!
49!!
50!! IMPLICIT ARGUMENTS
51!! ------------------

54!! JPVEXT : Vertical external points number

59!! XRD,XRV ! Gaz constant for dry air, vapor

67!! function over liquid water
68!! XALPI,XBETAI,XGAMI ! Constants for saturation vapor pressure
