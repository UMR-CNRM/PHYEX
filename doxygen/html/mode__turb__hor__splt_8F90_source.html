28!!**** *TURB_HOR* -routine to compute the source terms in the meso-NH
29!! model equations due to the non-vertical turbulent fluxes.
30!!
31!! PURPOSE
32!! -------
33! The purpose of this routine is to compute the non-vertical
34! turbulent fluxes of the evolutive variables and give back the
35! source terms to the main program.
36!
37!!** METHOD
38!! ------
39!! Complementary 3D calculations when running at high resolution;
40!! The non-vertical turbulent fluxes are computed explicitly. The
41!! contributions are cumulated in PRvarS and in DP and TP of TKE
42!
43! d(rho*T) = -d(rho*u'T'/dxx) -d(-rho*u'T'*dzx/dxx/dzz)
44! / dt / dx /dz
45!!
46!!
47!! Near the bottom of the model, uncentred evaluation of vertical
48!! gradients are required because no field values are available under
49!! the level where the gradient must be evaluated. In this case, the
50!! gradient is computed with a second order accurate uncentred scheme
51!! according to:
52!!
53!! D FF dzz3 (dzz3+dzz4)
54!! ---- = - ----------------- FF(4) + ----------------- FF(3)
55!! D z (dzz3+dzz4) dzz4 dzz3 dzz4
56!!
57!! dzz4 + 2 dzz3
58!! - ----------------- FF(2)
59!! (dzz3+dzz4) dzz3
60!!
61!! where the values are taken from:
62!!
63!! ----- FF(5)
64!! |
65!! | dzz5
66!! |
67!! ----- FF(4)
68!! |
69!! | dzz4
70!! |
71!! ----- FF(3)
72!! |
73!! | dzz3
74!! |
75!! ----- FF(2) , (D FF / DZ)
76!! | dzz2 * 0.5
77!! ----- ground
78!!
79!!
80!!
81!! EXTERNAL
82!! --------
83!! GX_M_U, GY_M_V
84!! GX_M_M, GY_M_M, GZ_M_M
85!! GY_U_UV,GX_V_UV
86!! GX_U_M, GY_V_M, GZ_W_M
87!! GX_W_UW,GY_W_UW
88!! : Cartesian vertical gradient operators
89!!
90!!
91!! MXM,MXF,MYM,MYF,MZM,MZF
92!! : Shuman functions (mean operators)
93!! DXM,DXF.DYM,DYF,DZM,DZF
94!! : Shuman functions (difference operators)
95!!
96!!
97!! IMPLICIT ARGUMENTS
98!! ------------------
99!! Module MODD_CST : contains physical constants
100!!
101!! XG : gravity constant
102!!
103!! Module MODD_CTURB: contains the set of constants for
104!! the turbulence scheme
105!!
106!! XCMFS,XCMFB : cts for the momentum flux
107!! XCSHF : ct for the sensible heat flux
108!! XCHF : ct for the moisture flux
109!! XCTV,XCHV : cts for the T and moisture variances
